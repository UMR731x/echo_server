# 🗺️ 终端地下城 (Terminal Dungeon) - 纯文本 RPG 冒险游戏

欢迎来到**终端地下城**！这是一个完全基于 Python 命令行界面（CLI）开发的复古文字冒险 RPG 游戏。

本项目**零外部依赖**，不需要安装任何第三方库。只需一行命令，就能在黑乎乎的终端里开启一场属于勇者的打怪升级之旅！

---

## ✨ 游戏特色

- **📦 零第三方依赖**：纯原生 Python 开发，有 Python 环境就能直接玩。
- **⚔️ 动态回合制战斗**：包含伤害随机波动、精英怪机制以及刺激的逃跑概率系统。
- **🛒 村庄补给商店**：可以用打怪赚来的金币购买红药水恢复生命，或者升级更高级的武器。
- **📈 完整的成长体系**：击杀怪物获得经验值，升级后自动提升生命值上限与基础攻击力。
- **💾 自动存档与读档**：利用内置 `json` 模块，游戏会在你胜利、失败或退出时自动保存进度，下次打开直接继续冒险。

---

## 🚀 快速开始

### 1. 准备工作
请确保你的电脑上已经安装了 **Python 3.x**。你可以在终端输入以下命令来检查版本：
```bash
python --version
```

### 2. 下载并运行
1. 下载或复制代码
2. 打开你的命令行工具（CMD、PowerShell 或 Terminal），切换到该文件所在的目录。
3. 运行以下命令启动游戏：
   ```bash
   python3 main.py
   ```
---

## 🎮 玩法指南

游戏主要由三个区域组成，通过输入数字编号（1-3）进行交互：

1. **⛺ 安全村庄（主菜单）**：在这里你可以查看自己的当前等级、血量、攻击力和金币，并选择接下来的行动。
2. **⚔️ 前往地下城（战斗）**：每次探索会随机遭遇不同强度的怪物。在战斗中你可以选择奋力一搏，或者在血量不足时尝试逃跑。
3. **🛒 铁匠铺与药店（商店）**：花金币购买红药水（恢复 50 点 HP）或升级武器（永久增加 10 点攻击力）。

> 💡 **小提示**：如果战斗失败被怪物击败，你将会失去身上**所有的金币**并在村庄里半血复活。记得在去地下城冒险前，先把金币花光变成装备哦！

## 📁 文件结构说明

运行游戏后，系统会在同级目录下自动生成一个存档文件：
* `mai.py`：游戏的核心源代码。
* `rpg_game_save.json`：自动生成的本地存档文件（技术原理：利用 Python 的 `json` 序列化功能）。**请勿手动修改该文件，否则可能导致存档损坏。**
